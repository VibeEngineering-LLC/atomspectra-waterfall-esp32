## 3. Скрипт и методология

### Протокол
Детектор AtomSpectra передаёт данные в бинарном **shproto-протоколе**:
- Фрейм: `0xFF 0xFE <data_escaped> 0xA5`
- Escape-байт: `0xFD` (инвертирует следующий байт)
- CRC16 (Modbus polynomial, таблица 256 записей)
- CMD `0x01` — порция гистограммы, `offset(2B) + counts(4B×N)`
- CMD `0x03` — текстовый ответ (на `-inf`, `-sta`)

Скрипт посылает команду `-sta` (старт потока гистограмм) и накапливает инкрементальную гистограмму
8192 каналов. Каждые **60 с** вычисляет разностную гистограмму (окно), находит **центроид** пика
в диапазоне каналов 1100–1900 (Cs-137 662 кэВ):

```
centroid = Σ(ch × counts[ch], ch=1100..1899) / Σ(counts[ch], ch=1100..1899)
```

Первые 5 окон — **BASELINE** (медиана → `baseline_median`). Последующие — **OK** если
`|Δ%| ≤ 2.0%`, иначе **ANOMALY**:

```
Δ% = 100 × (centroid − baseline_median) / baseline_median
```

Артефакты (нулевой центроид при разрыве USB) — пропускаются без записи.

### Параметры запуска
```
python stability_monitor_usb.py --port COM15 --window 60
    --ch-start 1100 --ch-end 1900 --anomaly-pct 2.0 --baseline-n 5
    --outdir scripts\stability_20260703_115514
```

## 5. Наблюдённые артефакты счётчика (не влияют на результат)

В процессе мониторинга через Monitor-задачу фиксировались резкие скачки `total_counts`
в отдельных окнах (~33 раза за 406 окон). Природа: **дублирование чанков гистограммы**
при чтении через USB — ESP32 шлёт порции данных, иногда одна порция считывается дважды.

**Ключевое:** центроид вычисляется как **взвешенное среднее** (числитель и знаменатель
масштабируются одинаково при дублировании) → дублирование чанков НЕ смещает центроид.
Это подтверждается: при ~33 флапах счётчика центроид оставался в норме, ни одного ANOMALY.

| Mean в Cs-окне (1100–1900) | ~80M counts/window |

## 8. Код скрипта (ключевые фрагменты)

### ShprotoParser — фреймер бинарного протокола
```python
class ShprotoParser:
    def __init__(self): self.buf=[]; self.started=False; self.esc=False
    def feed(self, b: int):
        if b == 0xFE: self.buf=[]; self.started=True; self.esc=False; return None
        if not self.started: return None
        if b == 0xFD: self.esc=True; return None
        if b == 0xA5:
            self.started=False
            if len(self.buf)>=3 and crc16(bytes(self.buf))==0:
                cmd=self.buf[0]; payload=bytes(self.buf[1:-2]); return (cmd, payload)
            return None
        if self.esc: b=(~b)&0xFF; self.esc=False
        self.buf.append(b); return None
```

### Вычисление центроида
```python
def centroid(histogram: list, ch_start: int, ch_end: int) -> float:
    total = sum(histogram[ch_start:ch_end])
    if total == 0: return 0.0
    return sum(c * histogram[c] for c in range(ch_start, ch_end)) / total
```

### Обнаружение аномалии
```python
delta_pct = 100 * (cen - baseline_median) / baseline_median
status = "ANOMALY" if abs(delta_pct) > args.anomaly_pct else "OK"
```
